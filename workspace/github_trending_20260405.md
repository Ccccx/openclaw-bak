- **🖥️ Neovim 集成**：提供 Neovim 插件（需 0.10.0+），键盘映射（`ff`/`fg`/`fz`）
- **💎 拼写容错**：极强的容错搜索能力
- **📊 调试支持**：可选的调试视图显示评分细节

#### 🎯 适用场景
AI 编码 Agent 提升文件搜索效率、Neovim 用户日常文件导航、大型代码仓库中的精准搜索

#### 🔮 趋势洞察
